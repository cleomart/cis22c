INSTRUCTIONS:

compile: javac Maze.java  AND javac Driver.java

run:
   If you want a randomly generated maze please type Driver.java n with n being the nxn dimension of your maze!

   If you want a file-read maze please type Driver.java 'filename' and make sure that the file is in the same directory as Driver.java!


   An incorrect run command will not work!
